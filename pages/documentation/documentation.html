To get started learning OData V4, we recommend working through the Beginner Tutorial and Advanced Tutorial first. Next, read Part 1: Protocol for an overview of the OData Protocol. See Part 2: URL Conventions for details on the URL syntax, and OData Common Schema Definition Language (CSDL) XML Representation for details on the OData Modeling language. Finally, refer to OData JSON Format for details on the OData JSON payload format.
