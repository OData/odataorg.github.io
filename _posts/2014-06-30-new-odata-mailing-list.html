In order to make OData.org a more collaborative site, we have maintained a Mailing List for people to ask or answer questions related to OData. And now we are moving to a new mailing list service OData Discussion Group.

  • For users already subscribed to our old Mailing List, please note that we will stop the support for it soon. So please move to the new OData Mailing List as soon as possible.
  • For new users, please just join the new mailing list.

You can view more detailed instructions of how to join the new mailing list for OData. All kinds of issues and feedbacks for OData are welcomed in this mailing list. If any problems and concerns, please comments below to let us know.
